====== 5 - Dynamic Programming ======
Created Thursday 10 October 2019

eg: Fibonacci Number

{{{code: lang="python" linenumbers="True"
def A(n):
	if n == 1 or n == 2:
		return 1
	else:
		return A(n-1) + A(n-2)
}}}
